Rain Man

Click to access a live demo.
View the GitHub repo

Rain Man is a web-application that allows you to view the current weather of any city in the world.
The app was built using Python, Django, SQLite, OpenWeatherAPI, HTML, and CSS.
This was my first solo full-stack project, with the goal of creating a responsive web-app with a simple, user-friendly UI.
